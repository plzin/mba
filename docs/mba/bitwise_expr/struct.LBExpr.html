Struct LBExpr

Source 
pub struct LBExpr<R: Ring>(pub Vec<(R::Element, BExpr)>);
Expand description

LBExpr is short for “Linear combination of Bitwise Expressions” These are the expressions for which rewrite rules can be efficiently generated.
